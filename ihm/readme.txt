hexagrille.html contient le code du plateau-test, qui a comme feuille de style hexagrille.css
La structure du plateau est générée grâce au script scritp_generate_board, les marbles ont été mises à la main en attendant d'avoir intégré ça dans le script
Les fichiers test et test2.xml sont des sorties obtenues avec le script précédemment cité


CONNEXION AVEC LE SERVEUR :
1. Ouvrir webserver.pl puis entrer Server(8000).
2. Ouvrir navigateur à l'adresse http://localhost:8000/game
3. Cliquer sur Transfert ! pour effectuer un test de requête (visible dans main.js). Cette dernière ne fait que retourner les paramètres qui lui sont envoyés

L'équivalent d'hexagrille est maintenant index.html
